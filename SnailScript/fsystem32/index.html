What is SnailScript?

SnailScript is a multi‑tier programming ecosystem designed for system programming, application development, web scripting, and machine‑level integration.

SnailScript Levels

WSS — Web SnailScript

Safe, sandboxed, browser‑friendly scripting. Extension: .wss

ASS — Application SnailScript

High‑level application development. Extension: .ass

SS — System SnailScript

Low‑level system programming. Extension: .ss

SSRM — SnailScript RawMode

Machine‑level integration. No safeguards. Sandbox recommended. Extension: .ssrm
